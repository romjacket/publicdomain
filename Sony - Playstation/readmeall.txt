psone-iso.zip
=============

The file "all.iso" contains the games:

Roller (beta)
PingPing
Robot Ron OPM
Ping

They were all written and designed by Matt Verran.

The ISO file has been written and checked with Easy-CD Pro 2.11 but
theoretically should would with other CD burning software so long 
as it support Mode2-XA mode burning.
